Galvanize Standard 3: A New Dr. Frankenstein?

My approach to building a blog from scratch was both exciting and frustrating. Starting a personal website has been on my to-do list for quite some time, so this was the perfect opportunity for me to buckle down and commit it. My initial plan was to take a template from online; however, my boyfriend let me know that that route wasn’t acceptable since I had the tools, knowledge, and resources to do it on my own. Especially since I love building things (and creating a site from scratch would help keep me out of “trouble” which for me is cleaning and organizing too much).

I’ve used a combination of HTML, CSS, and JavaScript to make this and have documented some of my notes as a learning log. One thing I do know is that my code is not particularly neat or concise. It’s pretty neat to think that I’ll be able to go back through the log of pushes to Github and see how my code has transformed over time. The biggest takeaway for me is that I can’t simply take a crash course in web development and then breeze through tasks like I have with singing and piano. I actually have to work for it and that is so incredibly satisfying.
